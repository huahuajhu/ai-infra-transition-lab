# Checkpoints

Use this folder for daily checkpoint answers.

Each answer should declare what kind of evidence it depends on:

- `Source-grounded`: the assigned reading/video/doc directly answers it.
- `Artifact-grounded`: the code, diagram, table, benchmark, or runbook you built answers it.
- `Synthesis`: the answer connects multiple concepts and may be revised later.
- `Prior experience`: the answer comes from your production background and should be revisited after more study.

Template:

```markdown
# Week N Day M Checkpoint

## Type
Source-grounded / Artifact-grounded / Synthesis / Prior experience

## Source or artifact
...

## Question
...

## My answer
...

## Confidence
High / Medium / Low

## Follow-up
...
```
